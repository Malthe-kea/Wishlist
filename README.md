# Wishlist

Dette er et program, der er lavet på datamatiker-studiet på KEA 2-semester efteråret 2024. 

Det er et mini-projekt kaldet _ønskeskyen_, hvor produktet er en hjemmeside, som gør det muligt
for en bruger at oprette en ønskeliste online med gaveønsker.

### Brugbarhed

1. **Gaveønskeren**
 * Kan oprette en bruger og ønskeliste med ønsker.
 * Kan dele sine ønsker.
2. **Gavegiveren**
* Kan få overblik hvad en person har af ønsker.
* Kan kordinere og se hvilke ønsker der er blevet købt.
